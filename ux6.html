WEB DESIGN


The Blind Pig is a responsive web design group project made with Bootstrap incorporating PHP for template design and Javascript for form validation.

The Blind Pig's original site can be viewed at www.blindpigmusic.com.
Note: Site design may have changed since the time of writing.

A personal portfolio was created for a class with HTML5/CSS3, Javascript, PHP, and Media Queries for responsiveness.

Both sites focused on:

  • Mobile-First Design
  • Accessibility
  • Responsiveness
